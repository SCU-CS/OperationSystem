## 自学指导

据说 **恐龙书** 和 **操作系统精髓与设计原理** 这两本很好，但文笔不太行，得硬挺着读下去，但 **OSTEP** 读起来就顺畅多了（而且还免费资源，何乐而不为），对某方面很感兴趣的话， **OSTEP** 还有很多推荐内容，可以慢慢看。

对Linux很感兴趣的话，可以看 **APUE** （Unix环境高级编程），厚得像是 Unix 的字典。

如果是应付考试的话，还是多看看学校那本书吧。
